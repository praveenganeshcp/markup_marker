Welcome to Markup Maker

Follow the below steps to quickly design wireframe with prebuilt widgets and generate code

  1. Click on the widget to select it as the parent widget. parent widget will be hightlighed in blue outline
  2. Click on one of the widgets from the Widgets list to append in the parent widget
  3. Right click on the widget to choose Edit, Copy, Delete options
  4. Finally click on the Generate code button to see the code output
